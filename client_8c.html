Detailed Description

File implementing the client side of the battleship game. The client connects to the server, sends the player's name, and then plays the game. The player is prompted to enter a move, which is then sent to the server. The game board is displayed after each move, showing the player's hits and misses. The game continues until all ships have been sunk or the server disconnects.
