Detailed Description

Structure for storing configuration options. The structure contains a key, a pointer to the value, and a function to parse the value. The key is used to identify the option in the command-line arguments. The value is a pointer to the variable where the parsed value will be stored. The parse function is used to convert the string value to the appropriate type.

Parameters
keyKey for the configuration option.
valuePointer to the variable where the parsed value will be stored.
parseFunction to parse the value.
